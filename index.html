Sobre o Projeto

O WhatsApp Web Clone é uma aplicação web progressiva (PWA) que replica as funcionalidades do WhatsApp Web, permitindo sincronização com uma conta existente do WhatsApp através de QR code.

Desenvolvido com Next.js e Tailwind CSS, o projeto oferece uma experiência de usuário idêntica ao WhatsApp Web original, com todas as funcionalidades esperadas de um cliente de mensagens.

A aplicação está preparada para uma futura integração com agentes automatizados para diferentes departamentos, permitindo a automação de atendimento ao cliente e outras interações.
